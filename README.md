
# HRMS Bonus Service

A simple Node.js service used to calculate employee bonus amounts based on salary slabs.

## Current Policy

- Salary < 500,000 → 2%
- Salary between 500,000 and 1,000,000 → 5%
- Salary between 1,000,001 and 2,000,000 → 8%
- Salary above 2,000,000 → 10%
